# Cpp My String

## Build

```
cd /path/to/sources
mkdir build && cd build
cmake ../
make
```

### Run example program using MyString

```
echo -e "aaa\nbbb\nccc\neee\nddd" | ./src/cpp_my_string
```

### Tests

```
./tests/tests -s
```

## Tested on compilers

```
* g++ (Ubuntu 4.9.1-16ubuntu6) 4.9.1
* Apple LLVM version 8.0.0 (clang-800.0.42.1)
```
